AI-Based Manufacturing Efficiency Dashboard

This project analyzes manufacturing machine performance using sensor, production, and network data. 
It includes KPI monitoring, efficiency classification, production trends, sensor analysis, correlation heatmaps, and AI prediction model.

 How to Run

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run app: `streamlit run app.py`

 Dataset

- manufacturing_dataset.csv included
